# PAAS平台建设清单

## 计划清单
- [x] 运营管理门户：业务运营系统
- [x] 任务调度平台：调度任务、定时任务、流程任务、脚本任务
- [x] 用户管理平台：组织、用户，内部、外部
- [x] 权限管理平台：内部业务系统功能权限管理，系统、角色、菜单、功能、分组、授权
- [x]  统一文件平台：文件上传、下载、管理
- [x]  统一流程平台：流程创建、发布、管理
- [ ] 核心交易平台：核心微服务
- [ ] 服务管理平台：服务注册、管理、熔断、限流、授权
- [ ] 商家管理门户：商家端业务
- [ ] 统一应用中心：应用、系统、权限
- [ ] 统一接口网关：认证、转发
- [ ] 统一配置平台：应用、环境、配置、历史记录
- [ ] 统一工单平台：业务工单、流程工单
- [ ] 统一部署平台：应用管理、发布部署、机器管理、灰度发布
- [ ] 统一日志平台：收集、查看
- [ ] 统一监控平台：机器、性能、业务
- [ ] 统一消息平台：短信、站内消息、微信消息
- [ ] 统一认证平台：Auth2、SSO
- [ ] 通用压测平台：接口、机器
- [ ] 通用测试平台：接口、消息
- [ ] 通用营销平台：有券、无券
- [ ] 通用数据平台：建模、查询、同步、数仓
